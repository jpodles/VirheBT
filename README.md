# About
VirheBT is a web-based  bug tracking system to assist development teams in project management and to track progress of fixing bugs or creating new features in IT projects.

# Tech Stack
- .NET 6
- Blazor Server Side
- ASP.NET 6
- MSSQL
- Blazorise
